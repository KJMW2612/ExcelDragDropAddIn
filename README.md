# Excel Drag & Drop Image Add-In 📊✨

Excel 365용 마우스 드래그 앤 드롭 이미지 자동 삽입 애드인입니다. 
폴더에서 이미지 파일을 마우스로 끌어 엑셀 시트 위로 떨어뜨리는 것만으로 지정한 셀 위치에 맞춤 리사이즈되어 삽입됩니다.

---

## 주요 기능 (Features)

* **드래그 앤 드롭 이미지 삽입**: 탐색기의 JPG, PNG, WEBP 파일을 마우스로 셀 위에 떨어뜨려 즉시 삽입합니다.
* **셀 크기 자동 맞춤**: 삽입되는 위치의 셀 너비와 높이에 이미지가 밀착하도록 자동으로 크기를 조절합니다.
* **셀과 함께 이동 및 크기 조절**: 열 너비나 행 높이를 변경할 때 이미지 크기도 셀을 따라 자동으로 변화합니다.
* **다중 파일 순차 배치**: 여러 개의 파일을 한 번에 드롭할 때 마우스 포인터 셀을 기준으로 아래 행 방향으로 줄을 맞춰 순서대로 정렬 삽입합니다.
* **정렬 조건 선택**: '추가 기능' 탭의 리본 메뉴를 통해 오름차순(기본값) 또는 역순으로 정렬 기준을 선택할 수 있습니다.

---

## 설치 및 사용 환경 (Prerequisites)

* **운영체제**: Windows 10 / Windows 11
* **Excel 버전**: 데스크톱용 Excel 365 (64비트 권장)
* **필수 런타임**: .NET Framework 4.8 이상

---

## 📥 다운로드 및 설치 방법 (Installation)

> ⚠️ **중요 (필독)**: 인터넷에서 다운로드한 보안 제약으로 인해 아래 **차단 해제 단계를 반드시 수행**해야 정상 설치됩니다.

1. 본 페이지 우측의 **[Releases]** 탭에서 가장 최신 버전의 **`ExcelDragDropAddin_v1.x.x.zip`** 파일을 다운로드합니다.
2. 다운로드한 ZIP 파일을 압축 해제하기 전, **파일을 마우스 우클릭 -> [속성]**으로 이동합니다.
3. 속성 창 맨 아래 보안 영역의 **`차단 해제 (Unblock)`** 체크박스를 체크한 후 **[확인]**을 누릅니다.
4. 압축을 해제한 후 폴더 내의 **`setup.exe`**를 실행합니다.
5. "게시자를 확인할 수 없습니다" 경고 창이 나타나면 **[설치]** 버튼을 눌러 설치를 완료해 주세요.

---

## 사용 방법 (How to Use)

1. 엑셀을 실행합니다.
2. 엑셀 상단 메뉴에 **[추가 기능]** 탭이 생성되었는지 확인합니다. 해당 탭 내에서 `이름 역순 정렬` 체크박스를 켜고 끌 수 있습니다.
3. 탐색기 폴더에서 올리고자 하는 이미지 파일(JPG, PNG, WEBP)을 다중 선택한 뒤, 엑셀 시트의 원하는 셀 좌표 위로 마우스를 끌어다 놓습니다.
4. 설정해둔 정렬 방식에 맞춰 이미지가 셀 높이와 너비에 맞게 정렬 삽입됩니다.
